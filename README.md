# Todo List Application

This is a simple Todo List application built with React. It allows users to add, edit, toggle completion status, and delete todo items.

## Features

- Add new todos.
- Mark todos as complete or incomplete.
- Edit existing todos.
- Delete todos.

## Technologies Used

- React
- HTML/CSS
